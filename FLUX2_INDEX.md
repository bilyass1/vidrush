## 📱 Quick Reference

### Essential Commands

```bash
# Start backend
cd apps/backend && npm run dev

# Start frontend
cd apps/web && npm run dev

# Run tests
export TEST_JWT="your-token"
ts-node test-flux2-service.ts

# Health check
curl -X POST http://localhost:3001/api/flux2/health \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### Essential Code

```typescript
// Import component
import Flux2ImageGenerator from '@/components/youtube/Flux2ImageGenerator'

// Use component
<Flux2ImageGenerator 
  onImageGenerated={(url) => console.log(url)}
/>

// Use API directly
import { flux2 } from '@/lib/api'
const formData = new FormData()
formData.append('prompt', 'A beautiful sunset')
const result = await flux2.generateImage(formData)
```
